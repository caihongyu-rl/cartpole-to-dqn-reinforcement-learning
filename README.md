# CartPole to DQN: A Reinforcement Learning Journey

## Project Goal
Implement and compare reinforcement learning algorithms from Q-learning 
to Deep Q-Network (DQN), trained on classic control environments 
(CartPole and LunarLander).

## Learning Path
MDP Basics → Q-learning → DQN → Policy Gradient → LunarLander Experiments

## Progress
- [ ] Phase 1: Q-learning implementation (Jun - Jul)
- [ ] Phase 2: DQN implementation (Aug - Sep)
- [ ] Phase 3: LunarLander core experiments (Oct - Nov)
- [ ] Phase 4: Algorithm comparison & documentation (Dec - Jan)

## Environment
- Python 3.13
- gymnasium (with box2d for LunarLander)
- PyTorch
- numpy / matplotlib
