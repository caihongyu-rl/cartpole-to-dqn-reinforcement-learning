# CartPole to DQN: A Reinforcement Learning Journey

## Project Goal
Implement and compare reinforcement learning algorithms from Q-learning 
to Deep Q-Network (DQN), trained on CartPole and Atari game environments.

## Learning Path
MDP Basics → Q-learning → DQN → Policy Gradient → Atari Experiments

- [ ] Phase 2: DQN implementation (Aug - Sep)
- [ ] Phase 3: Atari experiments (Oct - Nov)
